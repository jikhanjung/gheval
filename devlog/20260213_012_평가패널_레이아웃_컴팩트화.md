# 012: Evaluation 패널 레이아웃 컴팩트화

**날짜**: 2026-02-13

## 변경 요약

EvaluationPanel 높이를 줄이기 위해 세로 나열을 가로 배열로 재구성.

## 변경 내용

### GhComponents.py — EvaluationPanel.__init__()

| 영역 | 변경 전 | 변경 후 |
|------|---------|---------|
| 슬라이더 | 2×2 QGridLayout (숨김 포함) | QHBoxLayout 가로 배열 |
| Land Cover 컨트롤 | 결과 위에 가로 한 줄 | 좌측 컬럼 (Radius + Analyze 세로) |
| Land Cover 결과 | 5행×1열 세로 나열 | 2열×3행 (3+2) 가로 배치 |
| 라벨 | Dense Vegetation 등 긴 이름 | Dense Veg. 등 축약 |
| Notes | 높이 제한 없음 | maxHeight=60 |

레이아웃 구조:
```
Row 1: [Road Proximity ──────] [Vegetation Cover ──────]
Row 2: [Radius/Analyze | Dense Veg ██ 45%  Bare/Soil █ 15%]
       [               | Sparse Veg █ 30%  Built-up  █  5%]
       [               | Water      █  5%                  ]
Row 3: [Overall Risk] [Notes ─────────────────────────────]
```
